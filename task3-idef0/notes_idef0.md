# Конспект по изучению нотации IDEF0

## 1. Что такое IDEF0?

**IDEF0** - методология функционального моделирования для описания бизнес-процессов и систем.

## 2. Основные элементы

### Блок (Activity)
- Прямоугольник с названием функции
- Нумерация: A1, A2, A3...

### Стрелки (ICOM)
- **Input (Вход)** - слева (что преобразуется)
- **Control (Управление)** - сверху (правила, стандарты)
- **Output (Выход)** - справа (результат)
- **Mechanism (Механизм)** - снизу (ресурсы: люди, оборудование)

## 3. Декомпозиция

- **A-0** - контекстная диаграмма (1 блок)
- **A0** - диаграмма декомпозиции (3-6 блоков)
- **A1, A2...** - дальнейшая детализация

## 4. Правила

- 3-6 блоков на диаграмме
- Нумерация слева направо, сверху вниз
- Стрелки не должны обрываться

## 5. Отличия от BPMN

| IDEF0 | BPMN |
|-------|------|
| Функции системы | Поток работ |
| Нет временной последовательности | Четкая последовательность |
| 4 типа стрелок (ICOM) | Sequence Flow, Message Flow |
